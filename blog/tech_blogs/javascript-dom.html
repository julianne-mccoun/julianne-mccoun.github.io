Objects

An object is a type of data in Javascript which stores data in a structured way and can represent real world objects. To access and modify the data in an object, you use the object's properties. To declare a new object, you first use the variable keyword, and then type the object's name followed by the equals sign (=), and then an open curly bracket {. For each property you name the property and then put a colon and then the property value. This value must be in quotes if it is a string. Functions & why we need them

A function in Javascript can be simple or complex, it's all up to the developer! The basic purpose of a function is to group statements togeter to save us time and energy by allowing us to re-use our code! They also help us comply with the DRY principle in programming, which stands for Don't Repeat Yourself.

For example, if you wanted to print all of the names of your friends to your console, you could write a console.log() statement for each of your friends, and print it out that way one at a time:

What if we could make something to print out the names for us? This is where our functions come in! Below you can see an array of my friend's names, and a function I made called printNamess to print out the name of every object in an array for the given array that you indicate when you call the function. You can see the console messages on the right; they're the same as with the console.log() statements!

The great thing about functions is that now we can use this function to print out any array by calling it with the arrays name as the argument. Have a look at an example of this below where I printed out an array of my pet's names.

When creating a function, we declare the word function, the function's name, and then parenthesis after with any arguments for the function. To use our function we "call" it; to do this we simply write the name of our function on a line of code and any arguments for it in the parenthesis next to it. If there are no arguments needed for it, leave them blank. Our function will then run and return any output if it's included in the function.

This use of functions can be applied readily to many applications when you're writing a code. Functions are most useful when the code is repetetive and helps us define a block of code. Inside a function we can do things like declare variables, print console.log() statements, create loops and much more.
