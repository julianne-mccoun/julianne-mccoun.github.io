The Display Property

The

display:

property determines whether an element is treated as a block or as an inline element. Elements are defined by starting tags. Some elements have closing tags (such as the paragraph element) while others, called void elements don't need a closing tag (such as image elements).
Each type of element has a default display type, but these can be overwritten in your CSS file using

display: (preferred type)

. This will only change how it's displayed, not the kind of element. We'll discuss default display types a bit later.

display: inline

An inline element does not start on a new line, and it only takes up the necessary space.
Examples of inline elements include:

<span>

,

<a>

and

<img>

elements.
