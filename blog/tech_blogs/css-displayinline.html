CSS: Elements Displayed Inline versus as Inline-blocks

This blog post will explain what an HTML element is defined as, a brief explanation of what the display property

The Display Property and Elements

The display property determines whether an element is treated as a block or as an inline element. In HTML, elements are defined by starting tags. Examples of starting tags are the heading tags <h1> - <h6> Some elements have closing tags while others, called void elements, don't need a closing tag (such as image elements).

Each type of element has a default display value, but these can be overwritten in your code using the property display :. This will only change how it's displayed, not the kind of element. We'll discuss default display types a bit later.

Display: inline

An element displayed inline does not start on a new line, and it only takes up the necessary space.

Examples of inline elements include: <span>, <a> and <img> elements.
